About This Project

This application implements a Deep-Decision Fusion Framework for intelligent task scheduling using advanced machine learning classifiers:

  • PAC - Passive Aggressive Classifier
  • NBC - Naive Bayes Classifier
  • KNN - K-Nearest Neighbors
  • CRN-GRIM - Convolutional Recurrent Network with Greedy Rule Interpretable Machine

The system analyzes cloud workload data and predicts three key aspects: Job Priority, Scheduler Type, and Resource Allocation Type.
