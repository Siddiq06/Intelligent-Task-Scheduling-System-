# CRNGRIM: Intelligent Edge–Fog–Cloud Task Scheduling Framework

## 📌 Overview

CRNGRIM is an intelligent task scheduling framework developed using Python and Flask for Edge–Fog–Cloud computing environments. The system analyzes incoming tasks and determines the most suitable execution layer (Edge, Fog, or Cloud) based on task characteristics such as size, computational complexity, and resource requirements.

The framework combines machine learning algorithms with the CRIM-GRIM decision framework to optimize task scheduling and resource utilization.

---

## 🚀 Features

- Intelligent task scheduling across Edge, Fog, and Cloud
- Dataset upload and preprocessing
- Machine learning-based task classification
- Optimized resource allocation
- Interactive Flask web application
- Fast prediction and scheduling

---

## ⚙️ Workflow

1. Upload the dataset.
2. Preprocess the dataset.
3. Analyze task characteristics.
4. Classify tasks using machine learning algorithms.
5. Determine the best execution layer (Edge, Fog, or Cloud).
6. Display the scheduling result through the Flask web application.
